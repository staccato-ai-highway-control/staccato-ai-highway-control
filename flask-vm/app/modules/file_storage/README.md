# file_storage module

공통 파일 저장/검증 유틸을 둘 예정인 모듈입니다.

주의:
- 실제 파일 업로드 기능은 아직 구현하지 않습니다.
- app/api, app/services 디렉토리는 새로 만들지 않습니다.
- 기능별 API는 app/modules/<기능명>/routes.py에서 처리합니다.
- 공통 파일 저장 로직이 필요해지면 이 모듈에 유틸 함수로 분리합니다.

예정 역할:
- 파일 확장자 검증
- MIME type 검증
- 저장 파일명 생성
- 저장 경로 생성
- 파일 URL 생성
- 이미지/영상 타입 판별
