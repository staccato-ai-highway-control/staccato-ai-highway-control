# STACCATO 사용자 매뉴얼

## 1. 서비스 개요

STACCATO는 고속도로 CCTV 영상에서 정차 차량, 갓길 정차 등 이상 상황을 탐지하고 관제 화면에서 이벤트를 확인할 수 있는 AI 기반 도로 위험 탐지 시스템입니다.

## 2. 서비스 접속

브라우저에서 아래 주소로 접속합니다.

https://mbc-sw.iptime.org:3221

현재 시연/개발 환경에서는 자체서명 인증서를 사용하므로 브라우저에서 보안 경고가 표시될 수 있습니다.
내부 테스트 환경에서는 브라우저의 고급 설정을 통해 계속 진행할 수 있습니다.

## 3. 로그인

1. STACCATO 접속 페이지로 이동합니다.
2. 발급받은 계정 정보를 입력합니다.
3. 로그인 버튼을 클릭합니다.
4. 정상 로그인 시 관제 화면 또는 대시보드로 이동합니다.

## 4. 이벤트 목록 확인

이벤트 목록 화면에서는 AI가 탐지한 도로 이상 상황을 확인할 수 있습니다.

| 항목 | 설명 |
|---|---|
| 이벤트 유형 | 정차 차량, 갓길 정차 등 탐지 유형 |
| 발생 시간 | 이벤트가 감지된 시간 |
| 위치/ROI | 이벤트가 발생한 도로 영역 |
| 상태 | 이벤트 처리 상태 |
| 영상 | 이벤트 발생 구간의 MP4 영상 |

## 5. 이벤트 상세 확인

이벤트를 선택하면 상세 화면에서 다음 정보를 확인할 수 있습니다.

| 항목 | 설명 |
|---|---|
| 스냅샷 | 이벤트 발생 시점 이미지 |
| 이벤트 영상 | 이벤트 발생 전후 구간 영상 |
| AI 탐지 정보 | bbox, confidence, 이벤트 유형 등 |
| 처리 상태 | 이벤트 확인 및 관리 상태 |

## 6. 영상 확인 및 다운로드

이벤트 상세 화면에서 이벤트 영상을 재생하거나 다운로드할 수 있습니다.

영상은 HTTPS 환경에서 /ai-media 프록시를 통해 제공됩니다.
사용자는 별도의 AI-vm 주소로 직접 접근하지 않아도 됩니다.

## 7. 실시간 알림

Socket.IO 기반 실시간 알림을 통해 신규 이벤트 발생 여부를 확인할 수 있습니다.

| 상태 | 의미 |
|---|---|
| 연결됨 | 실시간 이벤트 알림 수신 가능 |
| 연결 안 됨 | 네트워크 또는 서버 연결 상태 확인 필요 |

## 8. 사용자 신고 업로드

사용자는 신고 업로드 기능을 통해 도로 위험 상황과 관련된 파일을 제출할 수 있습니다.

1. 신고 업로드 화면으로 이동합니다.
2. 파일을 선택합니다.
3. 필요한 정보를 입력합니다.
4. 제출 버튼을 클릭합니다.

## 9. 마이페이지

마이페이지에서는 사용자 계정 정보와 관련 기능을 확인할 수 있습니다.

## 10. 문제 해결

| 문제 | 확인 사항 |
|---|---|
| 접속이 되지 않음 | 접속 URL, 서버 상태, 네트워크 확인 |
| 보안 경고가 표시됨 | 자체서명 인증서 환경에서는 정상 |
| 로그인이 되지 않음 | 계정 정보 확인 |
| 이벤트 영상이 보이지 않음 | AI-vm 실행 여부와 AI media proxy 상태 확인 |
| 알림이 오지 않음 | Socket.IO 연결 상태 확인 |
